О сайте

Статичный сайт созданный для работы c API VimeWorld, на сайте можно просматривать глобальную и месячную статистику, а также расширенную по ограниченному списку мини-игр. Такую как: средний показатель по играм, сравнение показателей глобальной и месячной статистики. Благодаря rkkoszewski на сайте есть 3D визуализация скина и плаща игрока, с возможностью их скачивания.
